## Aroma Coffee

Навчальний сайт кав'ярні Aroma Coffee, створений для практичної роботи з HTML, CSS і JavaScript.

## Технології
- HTML5  
- CSS3 (змінні кольорів, адаптивність)  
- JavaScript (слайдер, форма)

## Кольори
#3e2723 (фон), #5d4037 (картки), #f5f5f5 (текст), #d4a373 (золотий акцент).

## Сторінки
- index.html – головна з акціями  
- menu.html – меню кави та десертів  
- aboutus.html – інформація про кав’ярню  
- location.html – адреси та карта  
- contacts.html – контакти й форма
